# Gold Prospectivity Mapping Using Multi-Criteria Evaluation (MCE)

![Gold Prospectivity Map](maps/gold_prospectivity_map.png)

## Project Overview
This project applies GIS-based Multi-Criteria Evaluation (MCE) to identify areas with high gold mineralization potential. It integrates multiple geospatial datasets including geological, structural, and surface features to generate a prospectivity map.

## Objective
To delineate zones of high and low gold prospectivity using spatial analysis and overlay techniques.

## Data Layers
- Hydrology and watershed model
- Topography Datasets including contours, hillshade and lineament models
- Hydrothermal Alteration datasets, including Argylic, Phyllic, and Propylitic Alteration Models
- Basemaps
- Vector Layers including artisanal mining locations and Title Area boundary

## Methodology
1. Data collection and preprocessing  
2. Band Ratio and combination of the multispectral datasets for Hydrothermal Alteration Mapping   
3. Hydrological analysis using the DEM to predict the hydrological model of the study area
4. Topographical processing to determine the relief of the study area
5. Fuzzy overlay analysis using the generated Hydrothermal Alteration datasets
6. Generation of final gold prospectivity map  

## Results
The analysis produced a gold prospectivity map highlighting:
- High potential zones   
- Low potential zones  

High prospectivity areas are associated with favorable lithology, as predicted by the combination of the properties of the ALteration datasets.

## Conclusion
This project demonstrates the effectiveness of GIS-based multi-criteria analysis in mineral exploration and decision-making.
